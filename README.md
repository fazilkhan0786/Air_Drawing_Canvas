# AirDrawing

Advanced hand-gesture drawing application using OpenCV.

## Project Attribution

This implementation is created by Fazilkhan Malek.

> Author attribution is included here in documentation and backend comments only. It does not alter the algorithm, the gesture controls, or the runtime behavior of the project.

## Overview

- Uses hand tracking and gesture recognition for drawing, erasing, thickness control, and clearing the canvas.
- The project is intended to run from `main.py` in the top-level project folder.
- The frontend behavior remains unchanged; this README simply documents the author.
